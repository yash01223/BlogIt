Overview

BlogIt is a RESTful blogging platform built using Java and Spring Boot that allows users to create, manage, and interact with blog posts securely. The application provides authentication, authorization, and CRUD functionality for blog posts while following a clean and scalable backend architecture.
The project demonstrates core backend development practices such as REST API design, JWT authentication, layered architecture, database integration, and API testing.

---------

Features

- User registration and authentication
- JWT-based authentication and authorization
- Create, read, update, and delete blog posts
- Secure API endpoints using Spring Security
- Layered backend architecture (Controller → Service → Repository)
- Global exception handling
- REST API testing using Postman
